## Começando

### Pré-requisitos

- Ultima versão do Node JS
```
    https://nodejs.org/en/download
```

### Instalando

1. Clone o repositório
```
    git clone https://github.com/ThiagoIanuch/Casa-do-Microondas.git
```

2. Instale os pacotes NPM
```
    cd client
    npm install
```

## Utilização
1. Inicie o servidor
```
    npm run dev
```

2. Acesse o site no navegador
```
    localhost:5173
```
